Parágrafos e quebras de linhas

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meu do par de tags p e /p

HTML é uma linguagem de marcação

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag br.

Se eu quiser fazer com que a tag br apareça então eu devo usar no lugar das setas os comandos <p> e </p> (veja no código), para que apareça a tag. Os comandos representam: menor que; e maior que; respectivamente.

Se você quiser fazer várias quebras de linhas n use <br> várias vezes, é feio.
É preferível usar o CSS.
